# node_upload_aws_s3
Compress JPEG, convert to WEB and upload both JPG/PNG and webp images to a AWS S3 Bucket

# Requirements
- Gm with webp support
- aws-sdk

# Install
```npm install```
or
``` yarn install```

# Replace with your own credentials
```
const ID = 'YOUR_ID'
const SECRET = 'YOUR_SECRET'
const BUCKET_NAME = 'BUCKET_NAME'

```

# Run
```node index```
